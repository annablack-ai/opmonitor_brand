模块 3: 排版与字号系统绝对隔离

Typography - CRITICAL. 所有设计文件必须区分数字屏幕端与纸质印刷端，两套参数不可混用。

数字屏幕端 (Web/UI)

适用于官网、H5、平台界面。重点是层级清晰与触控可读性。

  • H1: 40px - 56px, Bold, 行高 1.15
  • H2: 24px - 36px, SemiBold, 行高 1.25
  • Body: 16px - 20px, Regular, 行高 1.6
  • Caption: 12px - 14px, Medium

纸质印刷端 (Print/Brochure - A4尺寸)

适用于招商画册、方案集、招投标附件。严格遵循以下红线。

红线标准 (必须执行)

  • H1 封面主标题: 32pt - 42pt (加粗)
  • H2 模块小标题: 14pt - 18pt (中黑)
  • Body Text 正文描述: 8pt - 10pt (常规体, 行距1.5倍)
  • Caption 参数表格: 6.5pt - 7.5pt

特别提示: 这是画册呼吸感的生命线，正文字号绝不可超过 10pt，单段文字控制在 2-3 行。

模块 4: 空间留白与组件解构

Layout & Breathing Space. 画册版面需以留白建立秩序，负空间占比目标 50%，以降低医疗信息视觉疲劳。

禁令 1: 禁用粗边框

画册中慎用封闭描边卡片，改用留白或极淡无边框浅灰底（如 #F5F5F5）进行区域分组。

禁令 2: 数字降级

序列号（如 01, 02）不可使用巨大品牌蓝，建议用浅灰底纹或细线体，降低视觉噪音。

禁令 3: 强迫内边距 (Padding)

图文组合四周留白面积必须大于等于图文本身占据面积，强制形成呼吸带。

模块 5: 产品主视觉渲染规范

Hero Image. 所有硬件主视觉必须采用统一的白底/透明底策略，并叠加标准落地阴影，建立稳定、无菌、可信的设备质感。

强制说明: 严禁使用杂乱的医院实景背景。所有硬件（转运机器人、医废推车等）必须采用透明底/白底，并严格叠加标准深灰柔和落地阴影 drop-shadow(0 18px 16px rgba(38,45,58,0.22))，确保医疗器械的稳固感与无菌感。

模块 9: Icon 功能图标专业设计规则

Iconography Rules. 面向画册与 UI 的统一图标策略，确保医疗 B2B 语境下的专业感、可读性与克制表达。

核心禁令: 医疗 B2B 行业禁用复杂 3D、渐变色或拟物化图标。所有功能 Icon 必须保持高度抽象与克制。

  1. 1. 统一线宽 (Consistent Stroke): 统一采用 1.5px - 2px 极简线性描边，端点采用 Round Cap，传达医疗行业亲和力。
  2. 2. 双色调原则 (Dual-Tone): 主体线条使用深灰 #333333，关键点使用科技蓝 #007BFF 局部点缀，切忌全蓝或五颜六色。
  3. 3. 几何面性结合 (Micro-Flat): 可在图标底层加入极浅灰 #F5F5F5 几何托底，增加层次但不抢夺视线。
